# 투표 프로그램 사용설명서

학급 회장 선거, 반장 선거, 찬반 투표 등 다양한 상황에 사용할 수 있는 웹 기반 투표 도구입니다.

## 1) 시작 화면에서 준비하기
1. **투표 제목**을 입력합니다.
2. `-` / `+` 버튼으로 **투표 항목 수**를 조정합니다. (2~12개)
3. 각 항목 입력 칸에 이름을 입력합니다.
4. 우측의 **투표 시작** 버튼을 눌러 진행 화면으로 이동합니다.

## 2) 투표 진행 화면 사용법
- 안내 문구: **"하나를 선택한 뒤 오른쪽 아래 확인 버튼으로 투표를 완료하세요."**
1. 항목 하나를 선택합니다.
2. 항목을 선택해야만 **선택 확인 및 1표 반영** 버튼이 활성화됩니다.
3. 버튼을 누르면 `투표 완료` 오버레이가 약 1초간 표시됩니다.
   - 오버레이가 떠 있는 동안에는 다른 버튼 입력이 막힙니다.
4. 잘못 눌렀다면 **선택 초기화**로 선택을 취소합니다.

## 3) 투표 종료하기
- 우측 상단의 **투표 종료 및 결과 발표** 버튼을 누르면
  `투표를 종료하시겠습니까?` 팝업이 뜹니다.
- **취소**를 누르면 현재 투표를 계속 진행합니다.
- **확인**을 누르면 결과 화면으로 이동합니다.

## 4) 결과 화면 보기
결과 화면에서는 다음 정보를 한눈에 확인할 수 있습니다.
- 순위
- 득표 수 / 비율
- 가로 그래프(상대 비교)

## 5) 결과 화면의 두 가지 버튼
- **새 투표 준비**: 시작 화면으로 돌아가 새 항목/제목으로 다시 시작
- **다시 투표하기**: 직전 투표 제목/항목은 유지하고, 득표 수만 0으로 초기화한 뒤 투표 진행 화면으로 복귀

## 6) 모바일 효과음 안내
- 모바일 브라우저 정책 때문에 효과음은 사용자 첫 터치/키 입력 이후부터 안정적으로 재생됩니다.

## 7) 권장 환경
- 최신 Chrome / Edge / Safari / Firefox
- 태블릿, 노트북, 데스크톱 등 다양한 해상도 지원

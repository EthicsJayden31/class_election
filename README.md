# 투표 프로그램 사용설명서

학급 회장 선거, 반장 선거, 간단한 찬반/선호도 투표에 사용할 수 있는 웹 기반 투표 도구입니다.

## 1) 프로그램 소개
- 설치 없이 브라우저에서 바로 실행할 수 있습니다.
- 투표 항목 수를 정하고 이름을 입력한 뒤 즉시 투표를 시작할 수 있습니다.
- 결과는 순위와 그래프로 한눈에 확인할 수 있습니다.

## 2) 빠른 시작 방법
1. 페이지를 열고 **선거 제목**을 입력합니다.
2. **투표 항목 수**를 `+`, `-`, 숫자 입력으로 정하고 **칸 만들기**를 누릅니다.
3. 생성된 칸에 항목 이름(후보 이름)을 입력합니다.
4. **투표 시작** 버튼을 누릅니다.

## 3) 투표 진행 방법
1. 원하는 항목(후보)을 터치/클릭하여 선택합니다.
2. 하단의 **선택 확인 및 1표 반영** 버튼을 누르면 1표가 반영됩니다.
3. `투표 완료` 안내가 전체 화면 오버레이로 잠시 표시됩니다.
   - 이때는 다른 버튼이 눌리지 않아 오입력을 방지합니다.
   - 화면 아무 곳이나 터치하면 즉시 안내가 닫힙니다.
4. 필요하면 **선택 초기화**로 현재 선택을 취소할 수 있습니다.

## 4) 투표 종료 및 결과 보기
1. 투표 화면 우측 상단의 **투표 종료 및 결과 발표** 버튼을 누릅니다.
2. 카운트다운 후 결과가 공개됩니다.
3. 결과 화면에서 다음 정보를 확인할 수 있습니다.
   - 순위
   - 득표 수/비율
   - 가로 막대 그래프(상대 득표 비교)

## 5) 새 선거 시작하기
- 결과 화면에서 **새 선거 준비** 버튼을 누르면 설정 화면으로 돌아갑니다.

## 6) 권장 사용 환경
- 최신 Chrome, Edge, Safari, Firefox
- 태블릿/노트북/데스크톱 등 다양한 해상도 지원

## 7) 운영 팁 (교사용)
- 학생들이 줄을 서서 빠르게 투표할 수 있도록 기기를 고정해 두세요.
- 동일 화면에서 연속 투표할 때는 안내 오버레이가 사라진 뒤 다음 학생이 입력하도록 안내하면 실수를 줄일 수 있습니다.
- 선거 종료 전 **총 투표 수**를 확인해 참여 인원과 비교하세요.
